# Como Criar Ícones para PWA

## Opção 1: Usar Placeholder (Temporário)

Os ícones já estão criados como SVG. Para produção, você precisa:

1. Criar ícones PNG:
   - `icon-192.png` (192x192 pixels)
   - `icon-512.png` (512x512 pixels)

## Opção 2: Gerar Ícones Online (GRATUITO)

1. Acesse: https://realfavicongenerator.net/
2. Faça upload de uma imagem (logo VERUM)
3. Baixe os ícones gerados
4. Coloque em `/public/icon-192.png` e `/public/icon-512.png`

## Opção 3: Usar Ferramenta Online

- https://www.pwabuilder.com/imageGenerator
- https://favicon.io/

## Design Sugerido

- Cor de fundo: #10a37f (verde VERUM)
- Ícone: Sparkles ou símbolo VERUM
- Formato: PNG com transparência
- Tamanhos: 192x192 e 512x512

## Por enquanto

O PWA funciona mesmo sem os ícones PNG. O navegador usará o ícone padrão.
